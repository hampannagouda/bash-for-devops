# DevOps Playground

A hands-on collection of Bash and PowerShell scripts designed to build practical automation, Linux administration, and DevOps skills through real-world scripting exercises.

## Overview

This repository serves as a learning and practice environment for mastering scripting fundamentals and automation concepts used by DevOps Engineers, Cloud Engineers, and System Administrators.

The repository contains Bash and PowerShell scripts covering file management, package management, directory operations, user input handling, loops, functions, and system administration tasks.

## Technologies

* Bash Shell Scripting
* PowerShell
* Linux Administration
* Command-Line Automation
* DevOps Fundamentals
